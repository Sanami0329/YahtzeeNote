# YahtzeeNote

## 概要 
YahtzeeNoteは、Yahtzee（ヤッツィー）のスコアを紙要らずで記録できるアプリです。<br>
※Yahtzee：5つのサイコロを振って決められた組み合わせ（役）を作り得点を競うゲーム
* 簡単で素早いスコア入力、自動計算
* スコア履歴の保存、閲覧
* 一緒にプレイするメンバーの登録・編集
* 1人～最大6人まで同時プレイが可能
* レスポンシブ対応（スマホ・タブレット）
